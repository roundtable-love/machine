# واضح مواصلاتی پروٹوکول: 1.0

**حیثیت:** مکمل
**بنیادی زبان:** اردو (Urdu)
**قواعد:** سخت اور واضح تعریفیں

---

## 1. تعارف
یہ پروٹوکول واضح ارادے اور جسمانی اظہار کا بہترین امتزاج ہے تاکہ معلومات کا کوئی نقصان نہ ہو۔

## 2. بنیادی رابطہ: ماحول بنانا
* **وائب پنگ:** توجہ جانچنے کے لیے ایک سادہ سلام (جیسے "ہیلو")۔
* **ہم آہنگی:** جب دونوں افراد ایک ہی سوچ پر ہوں۔
* **پرسکون کرنا:** بات چیت کے لیے پرسکون ماحول بنانے کے لیے تناؤ اور انا کو ختم کرنا۔

## 3. غیر لفظی اشارے: جسمانی زبان
* **مکمل رکاوٹ:** ایک بڑا جسمانی اشارہ جو بات چیت کو فوراً روک دیتا ہے۔
* **مطابقت:** آپ کی جسمانی زبان آپ کے الفاظ سے ملنی چاہیے۔
* **اسٹاپ سگنل:** بڑی رکاوٹ کا مطلب ہے بات چیت ختم۔

## 4. خیالات کا ترجمہ: سننے والے کے مطابق ڈھلنا
* **بنیادی مطلب:** اصل ارادہ (لازمی، نہیں کرنا چاہیے، کر سکتے ہیں)۔
* **پیغام کو تبدیل کرنا:** سننے والے (ماہر بمقابلہ نیا) کی بنیاد پر بات کرنے کا انداز بدلنا۔
* **ذہنی دباؤ:** اگر سننے والا تھک گیا ہے، تو بات چیت روک دیں۔

## 5. کیس اسٹڈی: NVIDIA_HALT_2012
لینس ٹوروالڈز نے تعاون کی کمی کی وجہ سے "مکمل رکاوٹ" کا اشارہ استعمال کر کے NVIDIA کے ساتھ تعلقات ختم کر دیے۔

## 6. فن تعمیر
خیالات -> واضح قواعد -> سننے والے کے مطابق ڈھلنا۔

## 7. سخت قواعد
* **سیاہ اور سفید:** صرف "ہاں" یا "نہیں"۔
* **"چاہیے" نہیں:** صرف لازمی یا کر سکتے ہیں۔
* **زیرو لیک:** مطلب بالکل ایک جیسا رہنا چاہیے۔

## 8. میٹا ڈیٹا
* **زبان:** ur
